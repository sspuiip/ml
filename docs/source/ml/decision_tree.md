# 决策树

&emsp;&emsp;决策树是一种模拟人的决策过程中所用到的树形结构。例如：如果有以下一段对话，
>   女儿：多大年纪了？<br>
  母亲：26。<br>
  女儿：长的帅不帅？<br>
  母亲：挺帅的。<br>
  女儿：收入高不？<br>
  母亲：不算很高，中等情况。<br>
  女儿：是公务员不？<br>
  母亲：是，在税务局上班呢。<br>
  女儿：那好，我去见见。<br>

上述对话体现了人类做出见/不见决策的一个过程。根据对话的顺序，该决策过程可以用下图来刻画。
